TiltCheck Beta Installer

Quick Install (Recommended)

One-command installation for the complete TiltCheck ecosystem

curl -sSL https://tiltcheck.it.com/install.sh | bash

Docker Installation

Install using Docker Compose for containerized deployment

git clone https://github.com/jmenichole/TiltCheck-Ecosystem.git
cd TiltCheck-Ecosystem
docker compose up -d

Manual Installation

Step-by-step manual installation for advanced users

Development Setup

Setup for beta testers and developers

git clone https://github.com/jmenichole/TiltCheck-Ecosystem.git
cd TiltCheck-Ecosystem
npm install
cp .env.beta .env
npm start
